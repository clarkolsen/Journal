Browsing Vocabulary

9/14/21

Client: A web or other tool that displays web pages to a user.

Server: A computer online that stores web pages and delivers them to users.

Request: A message from a client /browser using a URL to ask a server for a web page.

Response: A message from the server that includes the code to show the web page.
